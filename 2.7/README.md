# 2.7 Declaring values

## Exercises: Heal the Sick

### 1

```haskell
let area x = 3. 14 * (x * x)
```

Answer: Space between 3. and 14.

```haskell
let area x = 3.14 * (x * x)
```

***

### 2

```haskell
let double x = b * 2
```

Answer: Variable 'x' not used in function declaration, instead 'b' is used.

```haskell
let double x = x * 2
```

***

### 3

```haskell
x = 7
 y = 10
f = x + y
```

**Answer:** All declorations must be on the same column.

```haskell
x = 7
y = 10
f = x + y
```
